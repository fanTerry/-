
###目录结构
<pre>
├── resources           // 资源文件存放
│   ├── images     	 	// 图片
│   ├── wxss         	// 公用样式文件
├── component        	// 公用组件存放
├── libs              	// 第三方插件包
├── pages         		// 项目所有页面,一个页面对应一个文件夹,以index文件夹为例,说明
	├── index     	 	// 首页文件夹
	  	├── index.js	// 必要,处理页面js
 		├── index.json  // 页面json数据
 		├── index.wxml	// 页面结构渲染
 		├── index.wxss	// 页面样式			
├── utils				// 工具类
├── app.js				// 整个项目的启动文件
├── app.json			// 整个项目的配置文件,比如注册页面,配置底部tab页,设置整个项目的样式,页面标题等
├── app.wxss			// 整个项目的公用样式文件
├── project.config.json // 项目配置文件,微信开发者工具创建项目的时候自动生成的,比如可配置,域名检测,es6语法支持,压缩,appid,projectname等
├── README.md			// 项目介绍
</pre>
